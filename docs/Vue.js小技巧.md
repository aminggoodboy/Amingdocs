## 去除 url 中的 `#`

**问题**：使用 Vue.js 搭建的项目，url 中会包含 `#`，影响观感。

**解决方法**：

1. 在项目中全局搜索 `const router = new VueRouter({})` 函数
2. 在函数内添加语句：`mode: 'history'`
