title: 宽带注入变压器的使用 🚧
---

宽带注入变压器用于将信号注入到电路中，例如开关电源的控制回路，配合环路分析仪实现开关电源的环路分析。本篇以 CYBERTEK 的宽带注入变压器 CK100（2Hz-2MHz）展开说明。

![](https://wiki-media-1253965369.cos.ap-guangzhou.myqcloud.com/img/20220513092658.png)

宽带注入变压器 CK100 的基本参数如下：

| 参 数          | 规 格            |
| -------------- | ---------------- |
| 比例系数       | 1:1              |
| 带宽（-3dB）   | 2Hz—2MHz         |
| 隔离电压       | 600V CATIII      |
| 初级次级间电容 | 400pF@1kHz       |
| 次级最大电流   | 10mA             |
| 工作稳定       | 0°C to 50 °C     |
| 尺寸           | 85mm ×46mm ×30mm |
| 重量           | 96g              |
