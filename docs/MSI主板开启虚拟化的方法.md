## 重启进入 BIOS

```cmd
shutdown.exe /r /o
```

重启后，点击 `疑难解答（Troubleshoot）` - `自定义设置（Advanced options）` - `UEFI 固件设置（UEFI Firmware Settings）`，进入主板 BIOS

## 找到相关设置

1. `F7` 进入高级选项
2. 依次点击 `OC` - `CPU 特征（CPU Features）`
3. 找到 `SVM Mode / Intel Virtualization（视 CPU 而定）`

## 修改设置

把 `Disabled（禁用）` 修改为 `Enabled（启用）`

## 保存退出

按 `F10` 保存并退出
