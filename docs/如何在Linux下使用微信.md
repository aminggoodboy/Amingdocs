DoChat（盒装微信）是一款适用于 Linux 桌面的容器化微信。

主要特色：

- 安装便捷，一行 shell 命令即可启动
- 支持汉字输入
- Ctrl+V 粘贴图片到微信

运行条件：Ubuntu Linux 桌面并安装 Docker
