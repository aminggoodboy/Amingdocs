## 基本操作

### 添加用户

```shell
useradd -m 用户名
```

### 设置密码  

```shell
passwd 用户名
```

### 删除用户

```shell
userdel  -r  用户名
```

### 删除用户目录

```shell
rm -rf 用户名
```

### 切换当前用户

```shell
su 用户名
```
