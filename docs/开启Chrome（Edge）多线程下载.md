## 如何开启

在地址栏输入链接并回车：

- Chrome：`chrome://flags/#enable-parallel-downloading`
- Edge：`edge://flags/#enable-parallel-downloading`

将 `Parallel downloading` 选项的 `Default` 改为 `Enabled` 即可。
